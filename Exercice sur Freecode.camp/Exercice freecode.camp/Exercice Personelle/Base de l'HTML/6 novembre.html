Apprendre le developpement web

Dans cette page web nous allons apprendre les différents langages de code : HTML , CSS , Javascript atribué au frontend

L'HTML

L'HTML ou hypertext market language est un langage mondialement utilisé pour la création de site web !

Le CSS

Le CSS ou Cascading Style Sheets est un langage courament utilisé pour l'interface visuelle d'un site web !

Le Java Script

Le Java Script est un langage de programmation responsable des animations des sites web!
